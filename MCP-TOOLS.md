
1. rust-mcp-server - Шорткаты для LLM (Экономия токенов за счёт выкидывания мусора из контекста)
Пример за место cargo clippy - получаем точеный json без мусора для LLM агента
https://github.com/0FL01/rust-mcp-server

2. tavily-mcp - Сервер для информации в интернете
https://github.com/tavily-ai/tavily-mcp

3. ssh-mcp-rs - Персистент ssh сервер для LLM агента, удобен, легковесен, прост (советую использовать авторизацию по ключам), можно сразу добавить несколько серверов и вызывать с помощью allias (агент поймёт какой это сервер)
https://github.com/0FL01/ssh-mcp-rs
